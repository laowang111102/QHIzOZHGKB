# 员工管理系统 java976

## 项目概述

员工管理系统是一款基于前后端分离架构的管理系统，主要面向企业内部员工信息管理需求。系统采用 SpringBoot 与 Vue 技术栈，实现了高效的员工信息管理功能。

## 技术栈

- **语言**: Java
- **前端技术**: Vue, Element-Plus
- **后端技术**: SpringBoot, Mybatis-Plus
- **数据库**: MySQL

## 功能模块

### 系统角色

- 管理员
- 用户

### 管理员功能

- 登录
- 首页
- 部门管理
- 员工管理
- 考勤管理
- 添加员工
- 奖惩管理
- 公告管理
- 合同管理
- 修改密码

### 用户功能

- 登录
- 首页
- 公告列表
- 合同信息
- 奖惩信息
- 考勤信息
- 个人信息
- 修改密码

## 运行环境

- Java 8 或以上版本
- MySQL 5.7 或以上版本

## 部署步骤

1. 准备 Java 和 MySQL 的运行环境。
2. 创建数据库，导入项目提供的 SQL 文件。
3. 编译后端代码，配置数据库连接。
4. 前端项目使用 npm 或 yarn 安装依赖，然后进行构建。
5. 部署前端代码到 Web 服务器。
6. 启动后端服务。
